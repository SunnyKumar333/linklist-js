# linklist-js
linklist-js is a npm package to perform linklist operation


## Types of LinkList

1. SinglyLinkList
2. DoublyLinkList

## Methods of SinglyLinkList
- Ading Element to SinglyLinkList
    1. addToHead(element): void
    2. addToEnd(element): void
    3. addAtIndex(element,index): void
- Deleting Element of SinglyLinkList
    1. removeFromHead(): element
    2. removeFromEnd(): element
    3. removeFromIndex(): element
- Other methods
    1. find(element): bool
    2. isEmpty(): bool
    3. print(): void

## Methods of DoublyLinkList
- Ading Element to DoublyLinkList
    1. addToHead(element): void
    2. addToEnd(element): void
    3. addAtIndex(element,index): void
- Deleting Element of DoublyLinkList
    1. removeFromHead(): element
    2. removeFromEnd(): element
    3. removeFromIndex(): element
- Other methods
    1. find(element): bool
    2. isEmpty(): bool
    3. print(): void
